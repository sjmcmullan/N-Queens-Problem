Parameters -> sa.py minN maxN PrintTheBoard
minN = The minimum board size you would like to run.
maxN = The maximum board size you would like to run.
PrintTheBoard = True or False, whether you want to see the printed solution for every n.

EXAMPLE:
>python sa.py 4 10 False
Run the script with 4 <= n <= 10 and you don't want to see every board printed.
